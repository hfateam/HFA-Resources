01 — Start Here
→ Three schools, one shared core +

These three California charter schools have the fullest applications in the set — but they ask for nearly all the same things. So you build the shared core once, then add a few school-specific extras for whichever you're applying to.

⚠ Use your own information throughout — this is a starting point, not a fill-in-the-blank. Most of the core comes straight from your standard packet, so have that ready before you start.
02 — The Shared Core
→ What all three ask for +
— Contact info: email, phone, website
— Description of services (your vendor blurb)
— W-9
— Proof of business legitimacy
— Certificate of Insurance
— DOJ background check / fingerprint clearance
— Custodian of Records (COR) option
— Delivery format: in person, virtual, or both

If your standard packet is built, you already have nearly all of this. For the background check and COR piece, lean on the background check / COR lesson.

06 — Keep in Mind
→ Before you submit +

Build the shared core once, then layer on each school's extras — you don't have to start from scratch three times.

For any of the named extras (TB clearance, Additional Insured, hold harmless, mandated reporter, secular certification), the Common Extras Explained reference has the plain-English version.

⚠ The Additional Insured Endorsement for Visions takes a few days to get from your insurer — request it early so it doesn't hold you up.
